# 🌾 KrishiMitra AI

> **AI-Powered Smart Agriculture Assistant**

KrishiMitra AI is an intelligent web application developed to help farmers make better farming decisions using Artificial Intelligence, Machine Learning, weather forecasting, crop recommendations, disease detection, government schemes, and multilingual voice assistance.

---

## 📌 Project Overview

Agriculture is the backbone of India, but farmers face many challenges such as:

- Wrong crop selection
- Unpredictable weather
- Fluctuating market prices
- Limited access to agricultural experts
- Language barriers
- Lack of digital farming support

KrishiMitra AI provides an all-in-one smart farming platform to solve these problems.

---

## ⚙ Installation

### Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/KrishiMitra-AI.git
```

### Open Project

```bash
cd KrishiMitra-AI
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Create .env File

```env
GEMINI_API_KEY=YOUR_API_KEY
```

### Run the Project

```bash
python app.py
```

Open:

```
http://127.0.0.1:5000
```
